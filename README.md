# pretix-betterpos

Native Pretix POS plugin for on-site sales with cash and euPago payment support.

## Status

Initial MVP implementation scaffold.

## Scope

- Event-scoped registers and cash sessions
- POS cart and order creation based on Pretix catalog
- Cash payments and async euPago payment initiation
- Auditable actions for discounts, cancellations, refunds, and cash movements
